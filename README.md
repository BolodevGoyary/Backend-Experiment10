# Backend-Experiment10: Simple REST API Endpoint

## Steps

### 1. Use Existing Database
Use the `backend_experiments` database and `posts` table from previous experiments.

### 2. Place Files
Put all files in:
MAMP: /Applications/MAMP/htdocs/experiment10  

Files:
- db_connect.php
- api_posts.php

### 3. Run API
MAMP:  
http://localhost:8888/experiment10/api_posts.php  

### 4. Test
Open the URL in a browser or use a tool like Postman/Insomnia to get JSON output of all posts.
